# Beautiful Applesauce

Beautiful Applesauce is the public teaser site and tester-link hub for Beautiful Apps.

## Purpose

- Present the brand with a simple landing page.
- Give testers a private `/links` page for alpha app access.
- Start with Count This as the first app listed for testers.

## Local Layout

- `index.html` - public landing page
- `links/` - unlisted tester-links page
- `assets/` - shared styles and illustration assets

## Publish Model

This project is intended to auto-publish from GitHub on push.
